# Luma 참가자 목록

Luma 이벤트 참가자 명단을 한눈에 보고 다운로드하는 Chrome 확장 프로그램

## 설치

1. 이 저장소 다운로드 (초록색 `Code` 버튼 → `Download ZIP`)
2. 압축 해제
3. Chrome 주소창에 `chrome://extensions` 입력
4. 우측 상단 **개발자 모드** 켜기
5. **압축해제된 확장 프로그램을 로드합니다** 클릭
6. 압축 해제한 폴더 선택

## 사용법

1. Luma 이벤트 페이지 접속
2. 화면 우측 상단 👥 버튼 클릭
3. 참가자 목록 확인
4. **복사** 또는 **CSV 다운로드** 버튼으로 내보내기

> 본인이 호스트이거나 참가자 목록 열람 권한이 있는 이벤트에서만 작동합니다.
